# Technical Stack

## Runtime Requirements

- PowerShell 7.0+
- Windows 10/11
- Git for Windows 2.40+

## PowerShell Modules

- z (directory jumper)
- posh-git (Git integration)
- Terminal-Icons (file type icons)
- PSReadLine (command line editing)

## Terminal Customization

- Oh-My-Posh (prompt customization)
- Windows Terminal (recommended)
- Custom theme: .mytheme-new.omp.json

## Azure Integration

- Az 10.0.0+
- Az.Accounts 2.12.0+
- Az.Resources 6.5.0+
- Az.Websites 2.12.0+
- Az.Functions 4.0.0+

## Git Configuration

- Multi-profile support via includeIf
- Directory-based email switching
- Productivity aliases

## Code Repository

- GitHub: https://github.com/mgpeter/usualexpat-devkit

## Deployment

- Local installation via PowerShell scripts
- Requires Administrator privileges for module installation
